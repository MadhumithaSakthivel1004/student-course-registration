# Student Course Registration System

## Overview
This is a Java console project to manage student enrollments in courses.  
I built it to practice OOP concepts, exception handling, and multithreading.

## Features
- Add students and courses
- Enroll and drop students
- Handle errors with custom exceptions
- Simulate concurrent registrations

## Tech Stack
- Java
- Collections Framework
- Multithreading

## How to Run
1. Compile the files:
   javac *.java
2. Run the program:
   java Main

## Sample Output
Student enrolled successfully!  
Course is full, cannot add more students.

## Learning Outcomes
- Improved my understanding of OOP
- Learned how to use custom exceptions
- Practiced multithreading in Java
